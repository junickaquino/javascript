What is AJAX?

- Asynchronous Javascript and XML
- Not a langauge, framework nor libraries. It's simpyl a set of web technologies to send and recieve data form a client to a server asynchrounously.
- XML is rarely used anymore, replaced my JSON. It's easier and faster.

- Lets you load chunks of a webpage rather than loading the entire webpage.

- XmlHTTPRequest (XHR) Object
    - API in the form of an Object
        - It has properties and methods attached to it.
    - Provided by browsers
    - Can be used other than HTTP
    - Can work with other data other than XML (ex: JSON, plain text)

- Libraries and Other methods
    - jQuery
    - Axios
    - Fetch API
